# MyBatis Learning

MyBatisの機能を学習するためのプロジェクト。Spring BootとMyBatisを使用したProject/Organization管理システムのCRUD操作を実装。

## Quick Start

```bash
# テスト実行
mvn test

# ビルド
mvn clean install
```

## Tech Stack

- Java 21
- Spring Boot 3.x
- MyBatis 3.5.16
- H2 Database (In-Memory)
- Maven
